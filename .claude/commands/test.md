# /test

プロジェクトの型チェックと必要に応じた動作確認を実行し、結果を報告します。

## 利用方法

```
/test
```

## 手順

1. `npm install` が済んでいるか確認し、必要なら実行します。
2. `npm run check` を実行して TypeScript の型チェックを行います。
3. 結果を要約して報告します。
   - 成功時: 全てのチェックが通過したことを伝えます。
   - 失敗時: エラー箇所と考えられる原因を整理し、修正方針を提示します。

## 注意

- 本プロジェクトに自動テストは含まれていません。`npm run check` が主な検証手段です。
- 画像の再生成 (`npm run generate`) は Gemini API のクオータを消費するため、`/test` では実行しません。
